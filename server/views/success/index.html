FAQ
The access to this application is restricted to some people, please contact an administrator at scipio@tircher.be, but don't expect a positive response if you don't know the application's administrators in real life, you've been warned !
If you received a code you either received a combination of a login and code or you asked for a specific login yourself, either way you can click on the "register" button at the top right of the application and enter both informations in the fields and then validate them by pressing the "validate" button. If they are correct your should be able to register your new account.
When an administrator creates a one-time code to allow a new account creation it will only be valid for a set amount of time. If you get this error double-check your informations but don't hesitate to contact an administrator to get your code regenerated.
If you have anymore questions don't hesitate to contact one of our administrator at scipio@tircher.be
